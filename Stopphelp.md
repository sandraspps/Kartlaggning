# Hjälp


## Så använder du appen

STOPP hjälper dig att lära dig om dina rättigheter.

Du kan:

•
Läsa och lyssna på lektioner

•
Gå i din egen takt

•
Gå tillbaka och läsa igen


## Navigering

Tryck på ett avsnitt för att se alla lektioner.

Tryck på en lektion för att läsa den.

Tryck på högtalaren för att lyssna.

Tryck på pilen för att gå tillbaka.
